## 简介
- 慕课上的[React16.4开发简书项目](https://coding.imooc.com/class/229.html)练习

## 运行
- cd 到项目目录下
  ```shell
  # 安装依赖
  npm i
  # 前端(http://localhost:3000)
  npm run start
  # 后端
  cd server
  node api
  ```

## 注意
- 登录时,账号密码任意
- `http://localhost:3000/todolist`是todolist练习项目
- `/build`下是打包完成的项目代码
